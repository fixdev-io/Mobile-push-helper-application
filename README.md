# Mobile Push Helper Application

A helper application for working with mobile push notifications.

## Tech Stack

- **Java:** 24
- **Kotlin:** latest version
- **Spring Boot**
- **Keycloak:** integration for authentication/authorization
- **Build:** Maven (`pom.xml`)

## Requirements

- **JDK 24** installed
- **Maven** (or Maven Wrapper, if included in the repository)

## Build & Run

### Build## Configuration

The application uses standard Spring Boot configuration (e.g., `application.yml` / `application.properties`).
Keycloak and environment-specific settings are configured via config files and/or environment variables.
